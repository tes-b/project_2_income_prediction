# project_2_income_prediction

## 프로젝트 개요
머신러닝으로 연봉 예측 모델을 만들고 연봉을 예측해 보는 프로젝트
  - 여러 국가와 회사의 연봉 데이터에 관한 통계 
  - 여러 선형 회귀 모델의 성능 비교
  - 파라미터 튜닝을 통한 모델 성능 향상
  - 완성된 모델로 지역, 경력, 학벌, 회사에 따른 연봉을 예측
